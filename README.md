# lxqt-themes-dracula
This repository houses the LXQt port of the <a href=https://draculatheme.com/>Dracula theme</a> by <a href=https://zenorocha.com/>Zeno Rocha</a>. It's one of the most popular and widespread themes in the world as of now, so why not have an LXQt version as well?
